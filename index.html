System Overview

How Skills Fit Into the Tier Formula

Technician tier placement is determined by a composite score across six weighted inputs:

InputWeight
Aptitude Assessment25%
Skills / Tags (ServiceTitan)10% of composite score
ServiceTitan Performance Metrics30%
Manager Score15%
Install Revenue10%
Google Reviews10%
Total100%
Important distinction: The Skills component contributes 10% to the composite score that determines tier placement. However, within a tier, accumulating skill tags governs level progression (C-1 → C-2 → C-3, etc.) and represents 30% of the weight for moving between tiers. You can earn more skills tags to climb levels within your tier, but you cannot advance to the *next tier* until your overall composite score clears the tier threshold.

Key Rules

  • No technician should be below C-2 as of current standing.
  • More skill tags = higher level within a tier. Level advancement within a tier is skills-gated.
  • Tier advancement requires both skill count AND composite score threshold.
  • Skills are assigned by the Service Manager. Techs do not self-assign.
  • Skills should be reviewed every 90 days or after a documented performance event.

Skill Tag Requirements Per Level

Skills tags are one of the three gates to advance levels within a tier. The tech must meet all three criteria:

Important: The skill count is a *gate*, not a guarantee. A tech can have 30 skills tagged but still be held at C-3 if the composite score hasn't cleared the B-tier threshold. Skills contribute 10% to the composite score and 30% of the weight for tier advancement decisions, but the full formula governs tier placement.
Updated: Skill count ranges have been adjusted upward to account for the expanded 48-skill system. Existing techs retain their current levels — no one moves backward due to the expanded skill list.

How Tier Advancement Works (Visual)

Within a tier:   Skill tags accumulate → Level rises (C-2 → C-3)
Between tiers:   Composite score must clear threshold (skills = 10% of that score)
AND manager approves the tier move
AND 30% of advancement "votes" come from skills progress

ServiceTitan Setup Instructions

Step 1: Access the Skills Settings

  • Log into ServiceTitan as an admin or manager.
  • Navigate to the main menu → Settings.
  • Under the Operations section, select Skills.
  • You will see a list of any existing skills. If starting fresh, the list may be empty.

Step 2: Create Each Skill

For each skill in the master list above:

  • Click + Add Skill (or "New Skill" depending on your ST version).
  • Enter the Skill Name exactly as listed in the "ServiceTitan Skill Name" column (e.g., Compressor Changeout).
  • Add a brief Description using the description from this document.
  • (Optional) If your ServiceTitan instance supports skill categories, group them using the category letters (A, B, C, D, E, F, G, H) or the full category name.
  • Click Save.
  • Repeat for all 48 skills.
Naming tip: Keep names consistent with this document. Avoid abbreviations that create ambiguity (e.g., use Evap Coil Replacement not ECR or Evap Swap). Consistent naming makes reporting and filtering clean.
New Category H: The Nexstar Service System Process skills (H1–H6) should be named with the "NSS —" prefix in ServiceTitan for easy filtering and reporting. Example: NSS — Greet, NSS — Explore, etc.

Step 3: Assign Skills to Technicians

  • Go to Settings → People → Technicians (or navigate to a tech's profile via the Team section).
  • Open the technician's profile.
  • Scroll to the Skills section (may be labeled "Tags" in some ST versions).
  • Click Edit or the + button to add skills.
  • Search for each skill by name and select it.
  • Save the profile.
  • Repeat for all 5 technicians using the assignments in this document.

Step 4: Link Skills to Job Types (Optional but Recommended)

ServiceTitan allows you to tag certain job types as requiring specific skills. This enables intelligent dispatching — the dispatch board will flag if an assigned tech doesn't have a required skill.

  • Go to Settings → Operations → Job Types.
  • Open a job type (e.g., "Compressor Changeout" or "System Replacement").
  • Find the Required Skills field.
  • Add the relevant skill(s) (e.g., A1 — Compressor Changeout, A7 — Refrigerant Management).
  • Save.

Step 5: Maintain and Review

  • Review skill assignments every 90 days or after a significant performance event (failed job, large install, customer complaint/compliment).

  • Use the ServiceTitan reporting suite to filter dispatching and performance data by skill to validate that skills assigned are producing the expected outcomes.
  • Revisit this document with each quarterly review cycle to add new skills as the team grows or service offerings expand.
  • NexTech Academy progress should trigger skill reviews: when a tech completes a NexTech level, schedule a ride-along or practical verification and assign the corresponding skills.

Appendix B: Nexstar Service System — 6-Step Quick Reference

This is the core customer interaction framework from Nexstar Network. Each step maps to a ServiceTitan skill tag (Category H). Use this as a coaching reference during ride-alongs and performance evaluations.

The 6 Steps

StepWhat the Tech DoesKey BehaviorsSkill Tag
1. PrepareMental and physical prep before arriving5-5-20 routine (5 gratitudes, 5 goals, 20 mins exercise). Check state of mind. Van/uniform inspection. Review job history in ST.Prerequisite (no separate tag — embedded in D3 SOP Compliance)
2. GreetCreate a great first impressionPark carefully, shoe covers, introduce self and company, empathy first, show agenda card, establish 3 pillars of trust (Empathy, Expertise, Expectations). No selling in the Greet — but sales can be lost here.H1 — NSS Greet
3. ExploreThorough investigation + customer engagementAsk symptom questions AND lifestyle questions. Keep customer engaged. Take photos. Fill out Summary of Findings. Price condition. Check in with dispatch. Prepare 3–6 options.H2 — NSS Explore
4. PresentShow options and let the customer decidePresent to ALL decision-makers. 3–6 options (A/B/C for HVAC). Each option includes: summary of work, warranty, financing, equipment, location, price, SPP info. Use "Link it, Say it, Spin it, Zip it" method.H3 — NSS Present
5. ExecutePerform the work with excellenceCraftsman quality. Leave area cleaner than found. Respect customer home. Manage time. Communicate to dispatch. No cutting corners.H4 — NSS Execute
6. Wrap UpClose the call professionallyTest all work. Show customer the completed repair. Review leave-behinds. Offer Service Partner Plan. Ask for 5-star review. Complete all paperwork. Perform curbside self-evaluation.H5 — NSS Wrap Up

3 Pillars of Trust (Established in Greet Step)

  • Empathy — Park carefully, use shoe covers, show concern for the customer's situation
  • Expertise — Ask a strong opening technical question, confirm intent to complete repair today, confirm Service Partner Plan status
  • Expectations — Visually show the customer the step-by-step agenda card process

Curbside Evaluation

After every call, the tech performs a self-evaluation at the curb before leaving:

  • Did I follow all 6 steps?
  • Did I present options to all decision-makers?
  • Did I offer the Service Partner Plan?
  • Did I ask for a 5-star review?
  • What would I do differently next time?
